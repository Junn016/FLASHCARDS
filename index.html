o que e o github?
O GitHub é uma plataforma online que funciona como um grande repositório de códigos de programação e arquivos, sendo uma ferramenta essencial para desenvolvedores e equipes que trabalham em projetos de software. Ele facilita a colaboração, o compartilhamento de código e a gestão de versões de projetos, através do uso do Git, um sistema de controle de versões distribuído.
